BachBase

Welcome to BachBase - a website designed to make it easier to find good quality classical music sheet music. If you have good quality sheet music (meaning readable and in modern notation), but you don't find it here, feel free to add the composer and/piece using the forms below. If you find that there is incorrect information, feel free to edit the composer or piece on the relevant page. Please note that most of the sheet music is linked to IMSLP, a non-profit sharing the world's public domain music. If you are using this website, please consider going to their website and donating to the project.
